European School Radio - Website


The European School Radio website was created in order to facilitate ordinary users to access and interact with the educational material available in the web platform of European School Radio.

The online platform of European School Radio, has been created with the content management system of WordPress which offers easy management of our website.

Primarily, the functions of the website, as they are adapted to special cases, have been written with our own PHP code. As the visual side has been created, for the most part, with the Visual Composer provided by Wordpress.

Overall, the European School Radio platform is a platform for all students, where everyone can interact with existing material as well as create their own with the aim of praising the transfer of knowledge to the learning community.

The main job of George Papadopoulos, was the maintenance of the already existing platform, as well as the development of additional parallel capabilities in the platform.

Project's Basic Information

  • Programming Language: PHP
  • Programming Environment: WordPress CMS
  • First Release: 2013
  • Project URL: European School Radio
